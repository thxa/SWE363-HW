Q2 (3 points): What do HTTP request and HTTP response contain?

Your Anwser:

A HTTP request contains:
  1. A request line.
  2. A series of HTTP headers, or header fields.
  3. A message body, if needed.
A HTTP response contains:
  1. A status line.
  2. A series of HTTP headers, or header fields.
  3. A message body, which is usually needed.
